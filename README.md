# GyroLedger
Warehouse Operations System (WOS), build in C# and .NET 10 using SQL Server as transactional and configurational data-storage.

## Major Sub-Systems
- Common Kernel
- Ledger Base and Rate Foundations
- System Meta-Hierarchy and Configuration
- Operational Model
- Item-Master Management
- Spatial Layout and Topology
- Item Ledgering
- Container Ledgering
- Order Management
- Aggregate Plan Management
